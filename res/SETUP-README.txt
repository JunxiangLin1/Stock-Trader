To run the jar file click the jar file.
Run the Jar file and then follow instructions below.

To create a portfolio type "create-portfolio <portfolio-Name>"
Then press enter
To add a stock type "add-stock-to-portfolio <portfolio-Name> <valid-Stock_name>"
Then press enter
Repeat until you have three stocks
To create another portfolio type "create-portfolio <portfolio-Name>"
Then press enter
Create two stocks with the same method as above
To query their value type "get-value <portfolio-Name> <valid-date in the format YYYY-MM-DD>"

Dates Supported are May 2012 - December 2024 all open stock dates
Stocks supported are All valid ticker symbols
